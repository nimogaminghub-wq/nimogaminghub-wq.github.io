Help & Support

Find answers to common issues and learn how to use the platform

📱 Android Games
  • Enable “Install Unknown Apps” before installing APK files.
  • Check Android version compatibility if installation fails.
  • Ensure enough storage before downloading large games.
🎮 PPSSPP (PSP Games)
  • Install the PPSSPP emulator before playing.
  • Use ISO or CSO game formats only.
  • Extract ZIP/RAR files before loading in emulator.
🕹️ PS2 Games
  • Use PCSX2 (PC) or AetherSX2 (Android).
  • PS2 games are large — ensure sufficient storage.
  • Configure BIOS and controls properly before playing.
⚠️ Common Problems
  • Download not working? Refresh and try again.
  • Game crashing? Adjust emulator settings.
  • Slow downloads? Check your internet connection.
  • File not opening? Make sure it is fully downloaded.
